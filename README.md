# Algoritmos e Estruturas de Dados - UniALFA 🎓

Bem-vindo ao meu repositório! Este espaço é dedicado ao armazenamento de atividades, exercícios e projetos práticos desenvolvidos na disciplina de **Algoritmos e Estruturas de Dados** da **UniALFA**. 

Todos os algoritmos e estruturas foram implementados utilizando a linguagem **C++**.

## 📂 Estrutura do Repositório e Conteúdos

Os arquivos estão organizados de acordo com os tópicos e avaliações propostas pelo professor. Abaixo estão alguns dos principais temas já abordados:

### Revisão N1
Pasta com exercícios fundamentais para a primeira avaliação (N1), colocando em prática conceitos-chave de C++:
- **Exercício 2 (Matrizes):** Lógica para encontrar e exibir a matriz transposta de uma matriz 5x4 populada aleatoriamente.
- **Exercício 3 (Funções com retorno):** Criação de função modularizada que retorna o dobro de um número.
- **Exercício 4 (Matemática básica em funções):** Função void para o cálculo de média aritmética de três números passados como parâmetros.
- **Exercício 5 (Sistema CRUD de Alunos):** Um sistema interativo rodando via terminal para gerenciar notas. Funcionalidades incluem:
  - **Create:** Cadastro de alunos e notas (limitado ao máximo pré-definido).
  - **Read:** Listagem completa e cálculo de Média Aritmética ou Ponderada.
  - **Update:** Alteração dinâmica de nomes e notas específicas.
  - **Delete:** Exclusão de registros do sistema.
  - **Manipulação de Arquivos (`.txt`):** Salvamento e carregamento automático dos dados (`alunos.txt`), garantindo a persistência das informações usando `<fstream>`.

## 🚀 Tecnologias Utilizadas

- **Linguagem:** C++
- **Ambiente/IDE:** CLion / GCC

## ⚙️ Como executar os projetos

Para compilar e executar qualquer um dos códigos, você precisará de um compilador C++ (como `g++`) instalado em sua máquina.

1. Clone este repositório:
   ```bash
   git clone https://github.com/SEU_USUARIO/SEU_REPOSITORIO_AQUI.git
   ```
2. Navegue até o diretório do exercício desejado:
   ```bash
   cd RevisãoN1
   ```
3. Compile o código desejado (exemplo do sistema CRUD):
   ```bash
   g++ exercicio5.cpp -o exercicio5
   ```
4. Execute o programa:
   ```bash
   ./exercicio5
   ```
